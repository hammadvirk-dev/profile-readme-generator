## 🚀 What is this?

A free, mobile-friendly web tool that generates professional GitHub Profile
READMEs using Claude AI. Just fill in your details, select sections, and
get a ready-to-use README.md in under 15 seconds.

## ✨ Features

- 🤖 AI-powered generation using Claude API
- 📱 Fully responsive — works on mobile, tablet & desktop
- 🎨 4 styles — Professional, Creative, Minimal, Fun
- 📊 GitHub Stats, Streak, Trophies, Snake Animation
- 💾 API key saved locally — enter once, use forever
- ⬇️ Download or copy with one click
